VoucherAPI - GenerateVouchersForCourse

Generate vouchers for a course.

Method signature

GenerateVouchersForCourse(string $email, int $amount, int $courseid [, array $groups = false [, bool $generate_single_pdfs = false]])

Parameters

nametypeoptional/mandatorydescription
emailstringmandatoryEmail address the vouchers will be sent to.
amountintmandatoryAmount of vouchers to be generated.
courseidintmandatoryID of the course the vouchers will be generated for.
groupsbooleanoptionalArray of IDs of all groups the users will be added to after using a Voucher.
generate_single_pdfsbooleanoptionalWill generate one PDF file for each voucher if true.

Return

typedescription
boolean$result

Example code / usage

 $url = 'http://moodle.menno.extern.ds.office.sebsoft.nl/blocks/voucher/view/api.php?method=GenerateVouchersForCourse&courseid=1&amount=5&email=menno@sebsoft.nl&groups[0]=1&groups[1]=2&generate_single_pdfs=1&resultType=xml';
 
 $params = array(
 'username' => '{API username}',
 'password' => '{API password}'
 );
 
 $ch = curl_init($url);
 curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
 curl_setopt($ch, CURLOPT_USERPWD, $params['username'].':'.$params['password']);
 $result = curl_exec($ch);
 
 echo htmlspecialchars($result);
